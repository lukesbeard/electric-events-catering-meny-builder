Tripleseat Proxy Example

How to Use the Proxy

This example demonstrates how to use the Tripleseat proxy server to avoid CORS restrictions.

Instead of calling the Tripleseat API directly, we call our proxy server, which forwards the request to Tripleseat and returns the response.

The proxy server URL in this example is: http://localhost:3000/api/tripleseat/leads

Update this URL to match your deployed proxy server.

Create Lead via Proxy

Code Example

// Client-side code to call the proxy server
fetch('http://your-proxy-server.com/api/tripleseat/leads', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    public_key: 'your_public_key',
    consumer_key: 'your_consumer_key',
    consumer_secret: 'your_consumer_secret',
    lead: {
      first_name: 'Test',
      last_name: 'User',
      email: 'test@example.com',
      phone_number: '555-123-4567',
      event_name: 'Test Event',
      description: 'Test Description',
      start_time: '2099-12-31 12:00:00',
      end_time: '2099-12-31 15:00:00',
      guest_count: 10,
      location_id: 'venue_id',
      room_id: 'room_id',
      status: 'new_lead',
      event_type_id: 'event_type_id',
      lead_source_id: 'lead_source_id'
    }
  })
})
.then(response => response.json())
.then(data => console.log('Lead created:', data))
.catch(error => console.error('Error:', error));
